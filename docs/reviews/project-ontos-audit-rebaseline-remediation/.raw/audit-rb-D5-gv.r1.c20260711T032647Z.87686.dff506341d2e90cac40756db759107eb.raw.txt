The `--print-timeout` flag is a command-line option in the **Antigravity CLI** (`agy`).

### Overview
* **Purpose**: It sets the timeout limit when running the CLI in **print mode** (non-interactive mode using `--print` or `-p`).
* **Default Value**: `5m0s` (5 minutes).
* **Format**: It accepts a standard Go duration string format (e.g., `30s`, `2m`, `1h`).

### Usage Example
When invoking a prompt via the CLI non-interactively, you can override the default 5-minute timeout like this:

```bash
agy --print "Write a summary of local git commits" --print-timeout 2m
```

If the agent does not finish executing and returning the response within the specified limit, the CLI process will timeout and exit.
