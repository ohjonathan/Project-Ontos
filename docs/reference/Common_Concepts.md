# Common Concepts

Standard vocabulary for tagging logs and documents.

| Concept | Description |
|---------|-------------|
| `api` | API design or implementation |
| `auth` | Authentication and authorization |
| `ci` | Continuous integration |
| `config` | Configuration management |
| `docs` | Documentation |
| `test` | Testing |
| `ux` | User experience |
